わけルンですv1.3.1

説明:
ディレクトリの参照を行うには右にある＋ボタンを押します。
実行に必要な参照ディレクトリは２つです。
サンプルディレクトリには自分が綺麗と思う写真を数枚入れる
ターゲットディレクトリには分類したい写真を入れる

デフォルトではresultファイルには分類結果が格納されます。

インストール方法?:
Matlabが入っていない方もしくは開かない方は Installer_web.exe を実行
わけルンです.exe を起動
※パスワードは担当の先生等に聞いてください。
